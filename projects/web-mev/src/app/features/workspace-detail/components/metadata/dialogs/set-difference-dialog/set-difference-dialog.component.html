To perform a set difference, the order matters. For example, consider the following:

  • set A with elements: "x", "y"
  • set B with elements: "x", "z"

If the order is (A,B), then the set difference contains only "y"; that is, "y" is the only element of set A that is not in set B.

If the order is reversed to be (B,A), then the set difference contains only "z"; that is, "z" is the only element of set B that is not in set A.
