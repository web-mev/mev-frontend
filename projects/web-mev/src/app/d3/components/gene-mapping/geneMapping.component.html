Note that there is often not a perfect one-to-one mapping between gene identifier systems. This is why we typically advise that users avoid switching between identifiers systems unless absolutely necessary. Examples where such operations might be necessary include gene pathway analyses where curated pathways may only be annotated in a single identifier system.

In the case where this re-mapping results in multiple "targets", we select the gene that has the largest median absolute deviation (MAD) with the assumption that variable expression profiles are likely to be more biologically "interesting" (e.g. in a differential expression context).
